"이걸 하면 나한테 좋은 건가?"

▶

사용자에게 가치와 혜택을 명확하게 전달해요

이점을 앞부분에 강조Don't AI 추천 시스템이 작동합니다
Do AI 추천으로 원하는 상품을 더 쉽게 찾을 수 있어요
구체적인 숫자 활용Don't 할인 쿠폰이 제공됩니다
Do 30% 할인 쿠폰을 받을 수 있어요
사용자 중심 언어Don't 오늘의집이 더 나은 경험을 제공하는데 도움됩니다
Do 관심사에 맞는 콘텐츠와 상품을 추천 받을 수 있어요
"하면 좋다"로 말해요Don't 이 기능을 사용해야 합니다
Do 이 기능을 활용하면 검색이 훨씬 쉬워져요!

"하려는 말이 뭐지?"

▶

명확하고 직관적인 문장을 작성해요

한 문장에 하나의 메시지Don't 앱 업데이트가 필요하며, 최신 버전을 사용해야 원활한 이용이 가능합니다
Do 앱을 업데이트해주세요. 최신 버전에서 원활하게 이용할 수 있어요.
군더더기 삭제Don't 보다 간편하게 결제하실 수 있어요.
Do 간편하게 결제할 수 있어요.
명확한 단어 사용Don't 사용자가 요청하면 해당 금액이 즉시 반환 처리됩니다
Do 즉시 환불됩니다.
단어 반복 검토Don't 배송이 시작되면 배송 알림이 발송돼요.
Do 배송이 시작되면 알려드릴게요.

"내가 뭘 하면 될까?"

▶

다음 행동을 명확하게 안내해요

다음 행동을 안내Don't 입력 정보가 올바르지 않습니다
Do 비밀번호는 8자 이상이어야 해요. 다시 입력해주세요
구체적 안내Don't 파일이 업로드되지 않았어요.
Do 지원되지 않는 파일 형식이에요. PDF 또는 JPG 파일을 업로드해주세요.
자연스러운 동기 부여Don't 업데이트하지 않으면 이용할 수 없습니다.
Do 업데이트하면 더 빠르고 안정적으로 사용할 수 있어요
피동형 지양Don't 결제가 처리되었습니다
Do 결제가 완료됐어요

"뭔가 찜찜한데...."

▶

이중 부정 질문 금지Don't 이 기능을 사용하지 않겠습니까?
Do 이 기능을 유지할까요?
긍정적 표현 사용Don't 진행을 원하지 않으세요?
Do 계속 진행할까요?
결정을 압박하지 않기Don't 이 기회를 놓치고 싶지 않죠?
Do 이 상품이 마음에 드시나요?
부정적 단어 지양Don't 인터넷이 끊겼습니다
Do 연결이 원활하지 않아요. 다시 시도해주세요

Do 마감이 얼마 남지 않았어요.

Do 이 제품을 구매하면 이런 혜택이 있어요

소리내어 읽고 마무리해주세요

텍스트가 자연스럽게 흐르는지 판단할 수 있는 가장 좋은 방법이에요.

새로운 용어와 표현을 쓰기 전, 히스토리를 확인해요

화면마다 다른 용어를 사용한다면, 사용자는 혼란을 겪습니다.

🧚🏻‍♀️
매끄러운 글쓰기 핵심TIP
논리적 모순과 번역투 지우기
  • 최상급의 모순 — 가장 오래된 것 중 하나 → 가장 오래된 것 / 오래된 것 중 하나
  • 영어식 진행형 지양 — 검토하고 있는 중이다 → 검토 중이다
  • ~에 대하여 → ~를, ~을
  • ~를 가지다 → ~를 갖다 / ~가 있다
  • ~에 대한 → 조사 '의'로 바꾸거나 생략
'의·적·것' 군더더기 걷어내기
  • '의' 삭제 — 사과의 나무 → 사과나무
  • '적' 자제 — 경제적 현상 → 경제 현상
  • '것' 줄이기 — 노력해야 하는 것이다 → 노력해야 한다
피동을 능동으로 바꾸기
  • 이중 피동 금지 — 생각되어집니다 → 생각합니다
  • 능동형 권장 — 제안이 받아들여졌다 → 제안을 수용했다
정확한 부호 사용하기
부호사용법예시
쉼표 ( , )열거하거나 흐름이 바뀔 때사과, 배, 복숭아를 샀다.
따옴표 ( " " )대화나 직접 인용에만그가 "안녕"이라고 말했다.
작은따옴표 ( ' ' )생각, 강조 용어'이게 맞을까' 고민했다.
가운데점 ( · )짝 단어, 밀접한 열거조사·분석
줄표 ( — )부가 설명 삽입그의 태도—무관심—가 화나게 했다.
단어의 올바른 호응
  • '결코, 전혀, 별로' + 부정어 — 별로 예쁘다 (X) → 별로 예쁘지 않다 (O)
  • '설마' + 의문문 — 설마 그가 올 것이다 (X) → 설마 그가 올까? (O)
겹말(동어반복) 삭제하기
  • 미리 예고하다 → 예고하다
  • 다시 재개하다 → 재개하다
  • 기존의 관례 → 관례
  • 따뜻한 온정 → 온정
모호한 '추측성 어미' 지양
  • 효율적인 방식인 것 같습니다 → 효율적인 방식입니다
  • 결과가 좋을 것으로 생각되어집니다 → 결과가 좋을 것으로 전망합니다

FAQ

▶
라이팅봇은 어떤 기준으로 문장을 수정하나요?

UX Writing 가이드를 기반으로 톤앤매너, 금지어, 문장 형식 등을 자동 검토해요.

꼭 GPT를 통해 문장을 써야 하나요?

선택 사항이지만, 브랜드 일관성 확보와 업무 효율을 위해 적극 권장해요.

우리 팀만의 스타일을 적용할 수 있나요?

가능해요. 목적별로 커스터마이징이 가능하며, 요청 시 팀 전용 라이팅봇 생성을 지원합니다.

Brand Language Design

브랜드 랭귀지 디자인은 브랜드의 정체성을 언어의 기준으로 설계해, 어디서 만나도 같은 브랜드로 느껴지게 만드는 일입니다.

❶ 추상적인 브랜드 가치를 말투·표현·문장 규칙으로 구체화해 브랜드의 Voice를 만듭니다.
❷ 프로덕트, 콘텐츠, CS까지 모든 접점에서 흔들리지 않는 한 가지 목소리를 유지합니다.
❸ 단순한 정보 전달을 넘어 브랜드의 태도가 드러나는 문장으로 신뢰를 쌓습니다.

UX Writing이 무엇인가요?

UX라이팅은 사용자 경험에서 언어를 다루는 모든 활동을 말합니다.

"서비스가 사용자에게 말을 거는 방식을 설계하고, 언어를 가꾸는 일"이라고 정의합니다. 버튼의 이름을 정하는 것부터 알림 메시지의 어조를 고민하는 일, 복잡한 정보를 쉽게 풀어 쓰는 일까지. UX 라이팅은 사용자가 서비스를 이용하는 전 과정에 함께합니다.

영역UX 라이팅카피라이팅테크니컬 라이팅
목적사용자 경험 향상설득, 마케팅기술 설명, 매뉴얼
형태짧고 간결한 UI 문구광고, 홍보 문구안내서, 긴 문서
사용자 중심필수선택적기술 중심

왜 UX Writing이 중요한가요?

▶

단어 하나, 표현 하나가 사용자 경험을 완전히 바꿀 수 있습니다.

BeforeAfter
[삭제][완전히 삭제할까요?]
[에러][사진 용량이 너무 커서 업로드할 수 없어요]
[확인][주문 완료하기]
[제출 완료][제출이 완료됐어요. 1~2일 이내 결과를 알려드릴게요]

UX Writer는 어떤 일을 하나요?

▶

📌 마이크로카피 작성 — 버튼 레이블, 안내 문구, 에러 메시지 등 UI에 직접 노출되는 짧은 문장들을 만듭니다.

📌 브랜드 말투 정립과 톤 앤 매너 유지 — 전체 서비스에 걸쳐 일관된 언어 톤을 유지할 수 있도록 스타일 가이드를 만들고 관리합니다.

📌 사용자 중심의 언어 설계 — 서비스를 처음 쓰는 사람도 막힘 없이 흐름을 따라갈 수 있도록 문장을 설계합니다.

UX writing 업무 구분

UX Writing 업무를 범위와 종류에 따라 구분한 기준입니다.

범위에 따라
Writing
문제 정의를 바탕으로 텍스트를 새로 작성하고 메시지를 구성
Editing
구조, 흐름, 정보 밀도를 포함한 내용 단위의 수정
불필요한 문장 제거, 순서 재정렬, 메시지 재구성
Refining
편집된 텍스트를 기준에 맞게 정밀 조정
논리·맥락·용어 일관성 점검
Polishing
배포 직전 단계의 표현 마무리
오탈자, 띄어쓰기, 뉘앙스 미세 조정
역할에 따라
Framework
글쓰기 기준과 규칙 정의, 템플릿 설계
Writing Guide · Template | Tone & Voice
Definition
이름과 개념을 정하는 일
Naming · Labeling
Production
실제 문구를 만드는 일
Component Writing | Transactional Content
Evaluation
기준에 맞는지 점검하는 일
UX Writing Audit
